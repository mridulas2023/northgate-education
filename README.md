# 📖 About

Northgate Education is a responsive educational consultancy website developed using **React** and **Vite**.

The platform is designed to provide students with everything they need for studying abroad—from university information and destination guidance to counseling support and AI-assisted interaction.

The website follows a premium corporate design inspired by leading international education consultancies with smooth animations, interactive sections, and modern layouts.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/northgate-education.git
```

## Navigate to Project

```bash
cd northgate-education
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Application will be available at

```
http://localhost:5173
```
